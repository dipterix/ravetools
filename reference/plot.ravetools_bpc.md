# Plot Basis Profile Curve results

S3 plot method for `ravetools_bpc` objects returned by
[`bpc`](https://dipterix.org/ravetools/reference/bpc.md). Draws a
three-panel figure: the basis profile curves, the subgroup-by-subgroup
significance matrix \\\Xi\\ ordered by assignment, and the per-subgroup
projection weights.

## Usage

``` r
# S3 method for class 'ravetools_bpc'
plot(x, ...)
```

## Arguments

- x:

  a `ravetools_bpc` object.

- ...:

  ignored.

## Value

Invisibly returns `x`.
